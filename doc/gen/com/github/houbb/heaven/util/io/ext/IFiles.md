## IFiles-文件接口

文件接口

### read-读取文件内容

读取文件内容
兼容性处理：如果 endIndex 超过文档的长度，则以文档长度为准

#### 方法入参

| 字段 | 说明 | 字段类型 |
|:---|:---|:---|
| startIndex | 开始下标 | 长整型 |
| endIndex | 结束下标 | 长整型 |

#### 方法出参

| 说明 | 类型 |
|:---|:---|
| 字节内容 | byte |
