### encodeToString-将字节数组转换成十六进制，并以字符串的形式返回

将字节数组转换成十六进制，并以字符串的形式返回
128位是指二进制位。二进制太长，所以一般都改写成16进制，
每一位16进制数可以代替4位二进制数，所以128位二进制数写成16进制就变成了128/4=32位。
